HOW TO USE THESE PATHS
================================================================================

1. Extract specific sprite sheets for your game engine
2. Use sprite sheet format (PNG with animation frame data in data.json)
3. Each sprite is pre-animated with multiple frames
4. Scale as needed for your game's screen resolution
5. Layer multiple sprites (e.g., Goblin + Explosion) for visual impact

IMPORT STRATEGY:
- Start with Viking-Escape main sprites (goblin warrior, archer, bomber)
- Add 2-3 flying enemies for variety
- Use Fire + Flame + Explosion effects as layered animations
- Add Zombie variants from Zombie-Buster for additional enemy types
- Use Coin + BonusBox for collectibles
- Add Blood effects for hit feedback
